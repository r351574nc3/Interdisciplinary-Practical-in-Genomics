class Gene

Description

Gene instances can be grouped together to form clusters (graphs of genes). A Cluster is basically like a digraph of genes where adjacent Gene instancess are grouped together. One Gene is known to be adjacent to another Gene if they belong to the same cluster (ie., have the same query id.) It is possible to union one Cluster with another. This would produce Gene instances that were adjacent because of common subject id

Author: Leo Przybylski (przybyls@arizona.edu)
