class Edge

Description

Edge instances can be grouped together to form clusters (graphs of genes). A Cluster is basically like a digraph of genes where adjacent Edge instancess are grouped together. One Edge is known to be adjacent to another Edge if they belong to the same cluster (ie., have the same query id.) It is possible to union one Cluster with another. This would produce Edge instances that were adjacent because of common subject id
