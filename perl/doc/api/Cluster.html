class Cluster

Description

A cluster is basically like a set edges in a digraph of genes where adjacent edges are grouped together. One Edge is known to be adjacent to another edge if ...

Being that a Cluster is a Set, there is no duplication

Author: Leo Przybylski (przybyls@arizona.edu)

Method add

Adds an Edge instance to the Cluster

Parameters

toadd - Edge instance to add

Method union

Unions this Cluster instance with another Cluster instance. The result is a completely new Cluster.

Parameters

other - a Cluster instance to union with this

Returns

A new Cluster instance containing all Edge instances from both Cluster instances.

Method contains

Parameters

tocompare -

Returns

1 if $tocompare is contained in the Cluster; 0 otherwise.

Getter edges

Returns

A reference to an array instance containing Edge instances for this Cluster

Getter size

Returns

The number of Edge instances that are part of this Cluster

Method compareCardinality

Compares the cardinality (the number of Edge instances) of this Cluster instance to another Cluster instance.

Parameters

tocompare - a Cluster instance to compare this against

Returns

-1 if this Cluster is smaller in cardinality than tocompare
0 if this Cluster is shares the same cardinality as tocompare
1 if this Cluster is larger in cardinality than tocompare
